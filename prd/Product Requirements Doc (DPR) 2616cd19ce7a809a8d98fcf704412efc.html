üìå Project Overview

üë§ User Personas

Who are you building this for?
  • Owner ‚Üí Pemilik platform, penyedia NFT khusus, audio, aksara Jawa, batik border, dll.
  • Partner ‚Üí Brand/komunitas/organisasi yang bisa buat NFT, set rules, kasih benefit, sebar QR code/radius minting.
  • User ‚Üí Pengguna umum, bisa mint, koleksi, upgrade, merge, dan nikmati benefit NFT.

üñ•Ô∏è User Stories

Written from the user's , owner, partners perspective.

by user prespective

  • as user
    • Aku ingin membuat NFT moment sehingga aku bisa menyimpan cuaca, waktu, lokasi, dan kondisi saat aku mint.
    • Aku ingin menambahkan border, sticker, font, filter, dan audio agar NFT-ku unik dan personal.
    • Aku ingin menggunakan aksara Jawa, kata-kata Jawa, gamelan, batik border supaya NFT punya identitas budaya.
    • Aku ingin scan QR code untuk bisa claim NFT yang disebarkan offline.
    • Aku ingin mint NFT dalam radius tertentu agar terasa eksklusif di lokasi tertentu.
    • Aku ingin meng-upgrade NFT untuk mengubah detail atau menambahkan elemen baru.
    • Aku ingin merge/fusion dua NFT supaya bisa menciptakan NFT baru yang lebih rare.
    • Aku ingin menyusun gallery NFT sehingga orang lain bisa melihat koleksiku.
    • Aku ingin punya timeline NFT untuk melihat perjalanan koleksi ku secara kronologis.
    • Aku ingin mendapatkan achievement jika berhasil kumpulin kategori tertentu (misal makanan + landmark).
    • Aku ingin mendapatkan benefit nyata (diskon, tiket event, hadiah) dengan mengoleksi kategori NFT tertentu.
  • as partner
    • Aku ingin membuat NFT sendiri dengan branding agar komunitasku bisa berinteraksi di platform.
    • Aku ingin menambahkan rules untuk NFT-ku (misal hanya bisa di-mint saat event, di radius tertentu, atau syarat scan QR code).
    • Aku ingin menetapkan kegunaan NFT (contoh: tiket konser, voucher makan, membership card).
    • Aku ingin memberikan benefit ke user yang mengoleksi kategori NFT tertentu.
    • Aku ingin menciptakan achievement mission agar kolektor lebih engage dengan NFT partner-ku.
  • as owner
    • Aku ingin menyediakan NFT special edition (dengan audio, aksara Jawa, batik border) untuk menarik user.
    • Aku ingin mengatur rarity system sehingga ada NFT yang lebih berharga/limited.
    • Aku ingin mengatur komponen kustomisasi (border, sticker, font, filter, audio) agar ekosistem terus fresh.
    • Aku ingin membuat event global dengan radius minting (misal di Candi Borobudur hanya bisa mint saat Waisak).
    • Aku ingin memungkinkan partner menambahkan NFT mereka sendiri agar platform lebih beragam.
    • Aku ingin memberikan reward/royalty dari transaksi NFT ke partner & creator.
  • cross role
    • Sebagai User/Partner/Owner, aku ingin NFT-ku bisa ditampilkan di gallery publik agar orang lain bisa melihat karya/koleksiku.
    • Sebagai User/Partner, aku ingin bisa verifikasi NFT via QR code atau blockchain explorer agar yakin keasliannya.
    • Sebagai User, aku ingin bisa share NFT-ku ke sosial media biar moment-ku dikenal.

by feature prespective

  • NFT Creation
    • Sebagai user, aku bisa membuat NFT untuk menyimpan moment pribadiku (foto/video/audio), sehingga moment itu tersimpan secara onchain.
    • Sebagai user, saat aku mint NFT, sistem otomatis menambahkan metadata cuaca, waktu, kecepatan angin, ketinggian, siang/malam, dan lokasi, sehingga NFT-ku unik dengan kondisi real-time saat mint.
    • Sebagai user, aku bisa menambahkan deskripsi atau judul pada NFT-ku, sehingga lebih personal.
    • Sebagai user, aku bisa memilih kategori NFT (misalnya makanan, event, landmark), sehingga NFT masuk ke koleksi tematik tertentu.
    • Sebagai owner, aku bisa membuat NFT khusus dengan konten premium (audio gamelan, aksara Jawa, quotes, dll) sehingga NFT memiliki value kultural.
  • NFT Customization (Border, Sticker, Filter, Font, Audio)
    • Sebagai user, aku bisa memilih border, sticker, font, sound, dan filter (seperti IG filter) untuk mempercantik NFT-ku.
    • Sebagai user, aku bisa membeli atau mendapatkan border/sticker khusus dari marketplace atau event tertentu.
    • Sebagai partner, aku bisa menyediakan border/sticker eksklusif untuk event/brand-ku, sehingga user bisa mint NFT dengan identitas event tersebut.
    • Sebagai owner, aku bisa menjual/airdrop border, font, sound eksklusif (misalnya batik border, audio gamelan, aksara Jawa).
  • NFT Reward, Benefit, & Rarity
    • Sebagai user, aku bisa mendapatkan NFT dengan rarity berbeda (common, rare, epic, legendary) yang mempengaruhi koleksi dan value.
    • Sebagai partner, aku bisa menambahkan rule khusus untuk NFT-ku (contoh: hanya bisa di-claim jika hadir di event, beli produk tertentu, atau menyelesaikan challenge).
    • Sebagai user, jika aku mengoleksi kategori NFT tertentu (misalnya 5 landmark dalam kota), aku bisa membuka achievement NFT atau benefit (diskon, voucher, akses VIP).
    • Sebagai user, aku bisa menukar atau redeem NFT tertentu untuk benefit nyata dari partner (contoh: diskon makanan, tiket konser, dll).
  • Location-based & QR NFT
    • Sebagai user, aku bisa scan QR code di lokasi tertentu untuk mendapatkan NFT eksklusif.
    • Sebagai user, aku hanya bisa mint NFT jika aku berada dalam radius tertentu yang sudah ditentukan oleh partner/owner (geo-fencing mint).
    • Sebagai partner, aku bisa membuat NFT yang hanya bisa di-claim di event offline dengan scan QR atau lokasi GPS.
    • Sebagai user, aku bisa melihat peta NFT (map) yang menunjukkan di mana lokasi-lokasi NFT bisa di-claim.
  • Upgrade & Merge NFT
    • Sebagai user, aku bisa meng-upgrade NFT milikku untuk mengubah detail tertentu (misalnya border baru, audio tambahan, rarity meningkat).
    • Sebagai user, aku bisa menggabungkan (merge/fusion) dua atau lebih NFT untuk membuat NFT baru dengan atribut gabungan.
    • Sebagai user, NFT hasil fusion bisa memiliki rarity lebih tinggi atau efek unik.
    • Sebagai partner, aku bisa mengatur syarat khusus untuk fusion tertentu (misalnya hanya bisa dilakukan dengan NFT event A + NFT makanan B ‚Üí unlock NFT spesial).
  • Gallery & Timeline
    • Sebagai user, aku bisa menampilkan semua NFT-ku di sebuah gallery pribadi yang bisa diakses publik atau privat.
    • Sebagai user, aku bisa melihat NFT-ku dalam bentuk timeline, sehingga koleksiku membentuk cerita perjalanan hidup atau pengalaman.
    • Sebagai user, aku bisa mengatur NFT ke dalam album/koleksi berdasarkan kategori atau event.
    • Sebagai user, aku bisa share gallery/timeline NFT ke sosial media dengan link unik.
  • Partner Features
    • Sebagai partner, aku bisa membuat NFT event/brand sendiri dengan rule custom (location, waktu, syarat tertentu).
    • Sebagai partner, aku bisa mengatur benefit NFT-ku (voucher, akses VIP, free gift).
    • Sebagai partner, aku bisa melihat analytics siapa saja yang meng-claim NFT event-ku (demografi, lokasi, engagement).
    • Sebagai partner, aku bisa membuat achievement series (misalnya koleksi 5 NFT event ‚Üí dapat NFT golden pass).
  • Owner Features (Platform-level)
    • Sebagai owner, aku bisa mengatur sistem rarity, supply NFT, dan komponen premium (aksara Jawa, gamelan audio, batik border).
    • Sebagai owner, aku bisa mengatur revenue share dengan partner dari penjualan NFT atau item kustomisasi.
    • Sebagai owner, aku bisa melakukan moderation (report NFT, banned user, validasi partner).
    • Sebagai owner, aku bisa membuat NFT eksklusif platform sebagai koleksi utama yang menjadi rare asset.
  • Social & Community Features (Opsional fase lanjut)
    • Sebagai user, aku bisa memberikan like atau komentar pada NFT teman.
    • Sebagai user, aku bisa trade NFT dengan user lain (P2P marketplace).
    • Sebagai user, aku bisa join komunitas berdasarkan kategori NFT tertentu (makanan, landmark, budaya).

How will we measure success?

Phase 1 ‚Äì MVP

üéØ Fokus: Proof of Concept & Core Experience

Target: User bisa mint NFT momen unik, Partner bisa buat drop sederhana, ada basic engagement.

Fitur untuk User (Core):

  • Social Login or Wallet Login
  • Mint NFT dengan metadata unik: lokasi, waktu, cuaca (ambil API cuaca real-time).
  • Customisasi sederhana: pilih border / sticker dasar.
  • Personal Gallery (timeline NFT moment, share sosmed).

Fitur untuk Partner (Basic):

  • Dashboard sederhana ‚Üí buat NFT drop (nama, deskripsi, cover, jumlah mint).
  • Rule dasar: location-based mint atau QR scan.
  • Benefit basic: diskon / voucher (code langsung muncul setelah mint).

Smart Contract / Infra:

  • NFT contract (ERC-721 / ERC-1155) dengan metadata.
  • Metadata hybrid (on-chain + IPFS).
  • Indexer (Ponder/Drizzle setup).

Phase 2 ‚Äì Growth

üéØ Fokus: Retention & Engagement

Target: User untuk pengalaman user,dan Partner bikin campaign kreatif.

Fitur untuk User:

  • Achievement/quest ‚Üí mint 7 hari berturut-turut = unlock batik border.
  • Rarity system (common, rare, legendary) ‚Üí bikin collectible feel.
  • Social sharing card (export NFT).

Fitur untuk Partner:

  • Partner rule:
    • hanya untuk user dengan koleksi tertentu
    • syarat join event / purchase product.
  • Analytics basic: jumlah mint, user aktif, top collector.
  • Multi-benefit: NFT bisa jadi tiket event, akses eksklusif, membership digital.

Smart Contract / Infra:

  • NFTCardModule, SpendingLimit (untuk daily cap misalnya untuk voucher).
  • Paymaster ‚Üí gasless minting untuk user.

Phase 3 ‚Äì Ecosystem

üéØ Fokus: Utility nyata + Ekosistem berkembang

Target: Web3 Moments yang dipakai brand, komunitas, event organizer.

Fitur untuk User:

  • Multi-card NFT (punya banyak NFT card dengan setting berbeda, daily limit, dsb).
  • NFT + NFC ‚Üí kartu fisik yang bisa dipakai buat check-in event / pembayaran.
  • Offline transactions (queue mint/claim walau tanpa internet ‚Üí broadcast saat online). (masih ragu gua)

Fitur untuk Partner:

  • Full dashboard ‚Üí drag-and-drop NFT campaign builder.
  • Custom reward system (point, cashback, loyalty tier).

Smart Contract / Infra:

  • NFT benefit on-chain ‚Üí programmable rules (NFT = akses gratis Wi-Fi 3 jam utility di dunia nyata)

Phase 4 ‚Äì Vision

üéØ Fokus: Scale & Monetization

Target: Jadi standar baru untuk ‚Äúmoment-based NFT + partner ecosystem‚Äù.

Fitur Monetisasi:

  • Subscription Partner ‚Üí bayar bulanan untuk akses dashboard campaign.
  • Transaction fee ‚Üí ambil % dari setiap mint/benefit claim.
  • Premium customization untuk user ‚Üí border/sticker premium.
  • Marketplace ‚Üí jual/beli NFT moment unik.

Ekspansi:

  • UMKM adoption ‚Üí NFT loyalty card digital.
  • Pemerintah / NGO ‚Üí subsidi, BLT, event edukasi berbasis NFT moment.
  • Global culture
